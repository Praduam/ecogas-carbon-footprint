# 🌱 EcoGas Carbon Footprint Awareness Platform

## Overview

EcoGas is a sustainability-focused web application that helps individuals understand, track and reduce their carbon footprint.

The platform calculates emissions from:

- LPG Consumption
- Electricity Usage
- Transportation
- Air Travel

Users receive:

- Eco Score
- Sustainability Badge
- Personalized Recommendations
- Interactive Dashboard

---

## Features

### Carbon Calculator

Estimate carbon emissions using:

- LPG cylinders
- Electricity consumption
- Car travel
- Flight frequency

### Eco Score

Users are awarded:

🌿 Green Champion

🌱 Eco Explorer

🌎 Climate Aware

⚠ Needs Improvement

### Dashboard

Interactive charts showing emission sources.

### Recommendations

Personalized actions to reduce emissions.
